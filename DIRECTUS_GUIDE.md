# Руководство по редактированию сайта через Directus

Directus — это админ-панель, через которую можно редактировать тексты, картинки, контакты и заявки на сайте **taltektrans.pro**.

## 🔐 Как войти

1. Откройте в браузере: `http://89.108.83.155:8055/admin`
2. Введите логин и пароль администратора (хранятся в файле `.env` на сервере, поля `DIRECTUS_ADMIN_EMAIL` и `DIRECTUS_ADMIN_PASSWORD`).
3. После входа интерфейс будет на русском языке.

> **Важно:** не передавайте логин и пароль третьим лицам. Если нужно дать доступ редактору — создайте отдельного пользователя с ограниченными правами.

---

## 🧭 Общая структура

В левом меню находятся **коллекции** — это блоки контента сайта. Коллекции бывают двух видов:

- **Singleton** — одна запись. Например, «Настройки сайта».
- **Обычная коллекция** — список записей. Например, «Ключевые услуги» или «Заявки».

### Основные обозначения полей

| Поле | Что означает |
|------|-------------|
| **Код** (`id`) | Уникальный идентификатор записи. Не меняйте без согласования с разработчиком. |
| **Статус** (`status`) | `published` — видно на сайте, `draft` — скрыто. |
| **Порядок** (`sort`) | Число, определяющее порядок записей (меньше — выше). |
| **JSON** | Поле с кодом. Обычно это список элементов в фигурных скобках. Меняйте текст, но сохраняйте структуру. |

## 🏠 Главная страница (`/`)

Отвечают коллекции:

- **Главная — Hero** — заголовок, подзаголовок и фон первого экрана.
- **О компании — вступление** — блок «О компании» на главной.
- **Наша миссия** — блок с галереей.
- **Ключевые услуги** — 3 карточки услуг.
- **Доставка — заголовок** — заголовок блока доставки.
- **Доставка — шаги** — 5 шагов доставки (01–05).
- **Почему мы** — преимущества компании.
- **Что вы получаете** — список выгод.
- **Цифры** — статистические показатели.
- **Парк вагонов — заголовок** — заголовок блока парка.
- **Парк вагонов — карточки** — цифры парка вагонов.
- **Модели вагонов** — карточки с вагонами.
- **Призывы к действию** — CTA-блоки на страницах.

### Как редактировать

1. Нажмите на коллекцию в меню слева.
2. Выберите запись (или единственную запись в singleton).
3. Измените текст или загрузите новое изображение.
4. Нажмите **Сохранить** в правом верхнем углу.

### Как поменять картинку

1. Нажмите на поле с изображением.
2. Нажмите **Загрузить файл** или выберите уже загруженное из библиотеки.
3. Сохраните запись.

> Рекомендуемый формат: `webp`, `jpg` или `png`. Размер файла не более 1–2 МБ.

## 📞 Страница «Контакты» (`/contacts/`)

- **Контакты — фон** — фон страницы.
- **Настройки сайта** — телефон горячей линии, адрес, email.
- **Контракты — заголовок** — заголовок блока заключения договора.
- **Контракты — контакты** — карточки контактных лиц.

- **Диспетчеры** — направления и телефоны.
- **Территории** — города и телефоны.

### Поля телефона

В коллекциях с телефонами обычно два поля:

- **Телефон** — текст, который видит посетитель (например, `+7 (495) 123-45-67`).
- **Ссылка на звонок** — техническая ссылка вида `tel:+74951234567` (без пробелов и скобок).

## 🔒 Страница «Политика конфиденциальности» (`/privacy/`)

- **Политика — вступление** — вводный текст.
- **Политика — разделы** — разделы политики. Поле **Блоки** (`blocks`) содержит JSON-массив. Каждый элемент может быть абзацем или списком.

Пример структуры:

```json
[
  { "type": "paragraph", "text": "Текст абзаца." },
  { "type": "list", "items": ["Пункт 1", "Пункт 2"] }
]
```

## ⚙️ Настройки сайта

Коллекция **Настройки сайта** (singleton):

- **Название сайта** — заголовок вкладки браузера.
- **URL сайта** — основной адрес сайта.
- **Логотип** — изображение в шапке и подвале.
- **Телефон горячей линии** — номер для отображения.
- **Ссылка на звонок** — `tel:+78005339888`.
- **Адрес офиса** — адрес в подвале.
- **Контактные email** — JSON-массив email-адресов.

## 📥 Заявки с сайта

Коллекция **Заявки** (`submissions`) содержит заявки из контактной формы.

- Поля: имя, телефон, email, компания, тип вагона, откуда/куда, комментарий.
- **Статус заявки**: `new` — новая, `processed` — обработана, `archived` — в архиве.
- Заявки сохраняются в Directus автоматически. Если Directus временно недоступен, заявка дублируется в файлы на сервере (`/root/taltek/data/`).

> Важно: email-уведомления о новых заявках сейчас могут не отправляться из-за ошибки авторизации SMTP. Для настройки email-уведомлений обратитесь к разработчику.

## 📝 Важные правила

1. **Всегда сохраняйте изменения** кнопкой в правом верхнем углу.
2. **Не меняйте поле `id`** без согласования с разработчиком — это может сломать сайт.
3. **Не удаляйте записи**, если не уверены. Лучше переведите статус в `draft`.
4. **JSON-поля**: меняйте только текст внутри кавычек, сохраняйте скобки, запятые и структуру.
5. **После изменений** сайт нужно пересобрать. Сейчас это делается вручную командой на сервере:
   ```bash
   cd /root/taltek && docker compose up -d --build frontend
   ```
   Либо настройте автообновление через веб-хук — уточняйте у разработчика.
